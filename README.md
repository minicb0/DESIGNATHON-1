## Run Locally

Clone the project

```bash
  git clone https://github.com/SANTHOSH17-DOT/DESIGNATHON.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Routes

#### Get home page

```http
  GET /
```

#### Get profile page

```http
  GET /profile
```

#### Get item page

```http
  GET /item
```

#### Get create item page

```http
  GET /create
```
